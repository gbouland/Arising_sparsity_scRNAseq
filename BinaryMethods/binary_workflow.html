Clustering workflow using binarized single-cell RNA-seq data

2022-11-01

This markdown is supplement to the paper: The rise of sparser single-cell RNA-seq dataset; consequences and opportunities. In this markdown we perform a complete clustering workflow using only binarized single-cell RNA-seq data. This workflow contains the following:

  • Finding a representative set of genes.

  • Clustering of cells.

  • Generating a UMAP plot.

  • Identification of marker genes.

  • Cell type annotation, using known marker genes.

Load Data

This single-cell brain dataset originates from Grubman et al. Cells without labels and doublets were removed, as well as cells originating from diseased individuals.

data <- readRDS("./data/exampleData.rds")
counts <- data[[1]]
samplesheet <- data[[2]]
rm(data)

Binarize Data

Binarizing is done by assigning a one to every non-zero value. Every zero remains a zero. The count matrix is removed, as it is not used for any downstream tasks.

binary <- binarize(counts)
rm(counts)

Find Representative Gene Set

repGenes <- binaryRepGeneSet(binary)

Clustering

For clustering, first the Jaccard cell-cell similarity matrix is calculated. Then, of this matrix, the eigen vectors are calculated. With these eigen vectors, using the euclidean distance, a shared nearest neighbor graph is constructed. This graph is fed to the Leiden clustering algorithm.

clusters <- binaryClustering(binary = binary[repGenes,], reduction = "JEV", nc = 15)

Make UMAP

For the UMAP, first the Jaccard cell-cell similarity matrix is calculated. Then, of this matrix, the eigen vectors are calculated. These eigen vectors are fed to the UMAP algorithm, again, using euclidean distance as metric.

UMAP <- binaryUMAP(binary = binary[repGenes,], reduction = "JEV", nc = 15)

Plot UMAP

plotData <- data.frame("UMAP1" = UMAP[,1],
                       "UMAP2" = UMAP[,2],
                       "cluster" = as.factor(clusters$cluster))
ggplot(plotData,aes(UMAP1,UMAP2, col = cluster)) + geom_point(size = 0.4, alpha = 0.7) + theme_minimal()

Find Markers For Each Cluster

Markers are identified using Pearson’s correlation coefficient between binary matrix and the one-hot encoded cluster assignment matrix.

markers <- findMarkers(binary,clusters = clusters$cluster)
